ConseilBox - Minimal Flutter scaffold generated by assistant.

## Prerequis
1. Flutter SDK 3.7+ ou compatible.
2. `flutter pub get`

## Lancer l'app avec l'API Laravel
Le client mobile lit l'URL de base depuis `ApiConfig` :
- Android emulateur -> `http://10.0.2.2:8000/api` (par defaut)
- iOS / simulateur / desktop / web -> `http://127.0.0.1:8000/api`

Pour cibler une autre machine (LAN, staging, prod), passez l'URL via un `dart-define` :

```bash
flutter run \
  --dart-define API_BASE_URL=http://192.168.1.23:8000/api
```

Meme principe pour un build release :

```bash
flutter build apk \
  --dart-define API_BASE_URL=https://api.conseilbox.com/api
```

> Astuce : gardez le port `:8000/api` si vous utilisez `php artisan serve`.
> Pour un serveur Apache/Nginx, donnez simplement l'URL publique complete.

## Structure
- `lib/config`: thème, couleurs, typo
- `lib/core/widgets`: composants partagés (navbar, cards…)
- `lib/features/home`: écran d'accueil
- `lib/features/settings`: paramètres / pages statiques

Font: google_fonts dependency (Delius) est déclaré dans `pubspec.yaml`.
