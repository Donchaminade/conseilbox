ConseilBox - Minimal Flutter scaffold generated by assistant.

How to use:
1. Install Flutter SDK 3.7+ or compatible.
2. Extract this archive.
3. From the project root run: flutter pub get
4. Run the app: flutter run

Structure:
- lib/config: theme, colors, text styles
- lib/core/widgets: shared widgets (navbar, card)
- lib/features/home: home screen

Font: google_fonts dependency (Delius) is declared in pubspec.yaml
